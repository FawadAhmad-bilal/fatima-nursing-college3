# Fatima Nursing College — Management System

A Firebase-backed student and fee management system built for a real nursing college.

## Features
- Student records: add, edit, delete, detailed profile view
- Fee management: log fees, generate receipts with receipt numbers
- Attendance: mark attendance, batch filtering, attendance reports
- Auto-generated student ID cards
- Live dashboard summary
- Toast notifications and modal-based UI throughout

## Tech Stack
- HTML, CSS, vanilla JavaScript
- Firebase (Firestore + Auth) as the backend — no server required

## Setup
Full step-by-step Firebase setup is included in `SETUP_GUIDE.txt` in this repo — covers creating the Firebase project, Firestore database, getting your config, and deploying free via GitHub Pages.

## Status
Built and deployed for a real nursing college's day-to-day student/fee/attendance tracking (not a tutorial project).
